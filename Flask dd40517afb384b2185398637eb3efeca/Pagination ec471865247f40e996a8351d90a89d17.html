Pagination

Loading all the post at once in single page makes our app slow. Hence we paginate.

Flask SQL-Alchemy offers paginate method to easily do this.

query.paginate() â€”> return Paginate object

This paginate object has following attributes and methods

per_page attribute returns the no. of results per page. By default it is 20.

page attribute returns which page we are in now

item attribute holds the result values

total attribute give the total no. of. results

we have seen on websites where if there are lot of pages the website shows at what page we are currently and then few pages before and after the current page with ellipses.

That is represented here as None over here.

Basically these numbers that is printed here are links to other pages.
